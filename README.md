# Machine_Learning_and_Medical_Devices
Heart disease prediction

The project aims to predict heart disease risk using machine learning techniques. The dataset contains various health indicators, and a Random Forest model is trained to identify individuals at high risk.

📂 Dataset
Source: Indicators_of_Heart_Disease_2022.zip
Preprocessing:
Missing values handled
Categorical features encoded
Features selected for modeling

⚙️ Technologies Used
Python 🐍
Pandas
Scikit-learn
Jupyter Notebook

🚀 How to Run
Install dependencies:
pip install pandas scikit-learn

Run the Jupyter Notebook:
jupyter notebook heart_disease.ipynb

📊 Model Performance
Algorithm: Random Forest Regressor
Metric: Mean Absolute Error (MAE)
